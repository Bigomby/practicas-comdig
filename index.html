Generar una secuencia aleatoria

Siendo \(N\) el número de muestras, para generar una señal aleatoria con valores \(\pm 2\) equiprobables:

An = 4 * randi([0 1], 1, N) - 2;

Podemos representar la secuencia con stem(An).

Cuando los valores no son equiprobables, el truco es usar la función sign(). Si las probabilidades están en la forma, por ejemplo, \(\{6/7, 1/7\}\), habría que generar 7 enteros, desplazar 6 hacia los números negativos (dejando uno en los positivos) y aplicar la función sign().

An = sign(randi([0 6], N) - 5.5);
Nótese que hemos restado 5.5 ya que si hubiésemos restado 6, hubiese habido un valor en 0 y el signo de 0 es 0. Sin embargo, esto puede ser de utilidad si quisiéramos generar tres valores con diferente probabilidad.

Función masa de probabilidad

Podemos calcular su función masa de probabilidad de la siguiente forma:

Código 1. Función masa de probabilidad
function [f, Xn] = LCDfmp(X)
  % Número de muestras
  Np = length(X);

  % Se descartan los valores repetidos
  valores = unique(X);

  % Se calcula el histograma
  [fx, Xn] = hist(X, valores);

  % Se divide por el número de puntos
  f = fx/Np;
end

Para representarla ejecutamos la siguiente orden:

Código 2. Representación de la función masa de probabilidad
[f, Xn] = LCDfmp(An);
stem(Xn, f);

Generar un pulso

Código 3. Generación del pulso en matlab
L = 8; (1)
sn = [a * ones(1,L/2), - a/2 * ones(1,L/2)];
1 Número de muestras

Para generar el pulso anterior, primero debemos calcular el valor de \(a\) para que la señal tenga energía unidad.

Calcular la energía de una señal
Cáclulo de la energía de una señal continua
\[E = \int_{-\infty}^{\infty} X(t)^2 dx\]
Cálculo de la energía de una señal discreta
\[E = \sum_{n=1}^{L} X_n^2\]

Aplicando Cálculo de la energía de una señal discreta sacamos el valor de \(a\).

\[1 = 4 a^2 + 4 \left ( \frac{a}{2} \right ) ^2 = 5a^2 \\ a = \sqrt{1/5}\]

Generar un pulso continuo a partir de uno discreto

Si queremos generar un pulso continuo a partir de s[n] que conserve la energía de la señal original y un intervalo de muestreo Tm lo haríamos de la siguiente forma:

\[s(t) = \frac{1}{\sqrt{T_m}} \cdot s[n]\]

Si nos dan el tiempo de bit (Tb) podemos calcular Tm de la siguiente forma:

\[T_m = \frac{T_b}{L}\]

L es el número de muestras.

Para generar la señal continua se haría de la siguiente forma:

t = 0:Tm:Tb-Tm; (1)
st = 1/sqrt(Tm) * sn; (2)
plot(t, st); (3)
1 Se define el intervalo donde estará definida la señal continua
2 Se generan los valores a partir de la señal discreta
3 Se dibuja

Cálculo de energía

Para calcular la energía de una señal discreta se puede hacer de la siguiente forma (hay otras):

E = sn * sn';

En el caso de una señal continua equivalente, sería:

E = Tm * st * st';

Cálculo de la función densidad de probabilidad

Para calcular la función densidad de probabilidad de una variable aleatoria usamos la siguiente función:

Código 4. LCDfdp.m
function [fxi,ci] = LCDfdp(X, xpuntos)
  % Número de muestras
  Np = length(X);

  % Número de puntos por defecto
  Npuntos=128;

  % Si se proporciona un segundo parámetro se usará en lugar de valor por defecto
  if nargin > 1
    Npuntos = xpuntos;
  end

  % Se genera un histograma
  [fxi, ci] = hist(X, Npuntos);

  % Se calcula la función aproximada
  fxi = fxi/Np/(ci(2) - ci(1));
end
Ejemplo 1. Cálculo de una FDP

Ejemplo de cálculo de la FDP de una normal con varianza 0.3 y media -3:

Np = 1e6; (1)
N = sqrt(0.3) * randn(1, Np) - 3; (2)

[fN, m2, v2] = LCDfdp(N,nvalues); (4)
plot(nvalues, fN); (5)

Modular una secuencia usando un pulso

Si tenemos una secuencia aleatoria y queremos usar un pulso conformador para modularla, lo ideal es unar la función kron().

Xt = kron(An, sn); (1)
stem(Xt);
1 An es la secuencia y sn es el pulso conformador
Ojo que el orden de los parámetros en la función kron() influye en el resultado.

Funciones densidad de probabilidad ponderadas

Tenemos:

  • \(X\) una variable aleatoria que toma valores \(\{+1, -1\}\).

  • \(N = N_1 + N_2\). Siendo \(N_1\) y \(N_2\) son dos variables aleatorias uniformemente distribuidas entre \([-1, 1\)].

  • \(Y = X + N\)

Representar en una misma gráfica:

  • \(q \cdot f_{Y \mid X = -1}(y \mid X = -1)\)

  • \(p \cdot f_{Y \mid X = 1}(y \mid X = 1)\)

Siendo \(p\) y \(q\) \(\{1/4, 3/4\}\) respectivamente.

Encontrar su frontera

Para encontrar la frontera, es decir el punto en el que se cortan \(f_{Y \mid X=-1}\) y \(f_{Y \mid X=1}\) podemos usar la función find() de la siguiente forma:

idx = find((fY1 <= fY2) & (fY1 > 0), 1); (1)
g = r(idx); (2)
1 En idx tenemos el índice del punto donde se cortan las funciones
2 En g tendríamos el valor de las funciones en ese punto

Siendo \(M\) el número de símbolos, \(K\) es el número de bits que corresponden a cada símbolo.

\[K = log_2 M\]

Transmitir una señal PAM

  • Parámetros

    • Bn: Vector de bits a transmitir.

    • Eb: Energía de bit.

    • M: Número de símbolos.

    • p: Depende de \(g(t)\):

      • \(g(t)\) es paso de baja: \(p = g(t)\)

      • \(g(t)\) es paso de banda: \(p = g(t) \cdot cos(\omega_c t)\)

    • L: Número de muestras en la señal discreta.

  • Retorno

    • Xn: Seañal transmitida.

    • Bn: Lo mismo que se introdujo, pero relleno con ceros si es necesario.

    • An: Vector de símbolos a transmitir.

    • phi: Pulso conformador: Depende de \(g(t)\):

      • \(g(t)\) es paso de baja: \(\phi(t) = \sqrt{\frac{1}{E_g}} g(t)\)

      • \(g(t)\) es paso de banda: \(\phi(t) = \sqrt{\frac{2}{E_g}} g(t) \cdot cos(\omega_c t)\)

    • alfabeto: Vector con los posibles valores de los símbolos.

Código 7. Transmitir la señal
K = log2(M);
Es = 5;
Eb = Es/K; (1)
[Xn, Bn, An, phi, alfabeto] = transmisorpam(Bn, Eb, M, p, L);
1 Recordemos que la energía de bit es la energía de símbolo entre el número de bits por cada símbolo (K).
p debe ser una señal discreta, por lo que \(\omega_c t\) pasa a ser wc * n, siendo n = 1:1:L.
Representar n símbolos de la señal transmitida
t = Tm:Tm:L*Tm*N;
plot(t, Xt(1:N*L));

Siendo N el número de símbolos a representar

Función transmisor PAM

function [Xn,Bn,An,phi,alfabetopam] = transmisorpam(Bn,Eb,M,p, L)
  % ¿Cuántos bits hay en cada símbolo?
  k = ceil(log2(M));

  % Ajustemos M a una potencia de dos
  M = 2^(k);

  % El alfabeto [Ver la ecuación (4.10)]
  alfabetopam=sqrt(3*Eb*log2(M)/(M^2-1))*(2*(1:1:M)-M-1);

  % Si la secuencia Bn no tiene una longitud múltiplo de k, se completa con
  % ceros
  Nb = length(Bn); (1)
  Bn = [Bn, zeros(1, k*ceil(Nb/k) - Nb)];
  Nb = length(Bn); (2)
  Ns = Nb/k; (3)

  % La secuencia generada
  if M > 2
      An=alfabetopam(gray2de(reshape(Bn,k,Nb/k)')+1);
  else
      An=alfabetopam(Bn+1);
  end

  % Comprobación de las longitudes y otros datos del pulso básico para hacer
  % que el número de muestras del mismo sea efectivamente L
  Ls = length(p);
  if Ls<L
      p = [p,zeros(1,L-Ls)];
  else
      disp(['La duración del pulso se ha truncado a ',num2str(L), 'muestras'])
      p = p(1:L);
  end

  % Normalicemos la energía del pulso suministrado para obtener la base del
  % sistema
  phi = (1/sqrt(p*p')) * p;

  % Obtención del tren de pulsos
  Xn = kron(An, phi);
end
1 Número de bits que vamos a transmitir
2 Número de bits que vamos a transmitir tras la corrección
3 Número de símbolos que vamos a transmitir
